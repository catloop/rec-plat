接入流程

  • 注册账号

    注册个人或企业支付宝账号，入驻成为开发者。

    1

  • 完善资料

    注册后，您需要完善个人资料，绑定邮箱和手机号。
    您只有绑定帐号后，才可以进行服务购买、 SDK 下载以及 API 接入。

    2

  • 获取授权

    前往“价格方案”，生成订单或联系“技术支持”，进行线下购买。

    3

  • 开始接入

    创建 API Key ，下载 SDK ，阅读文档，开始编码。

    4
